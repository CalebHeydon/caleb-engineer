Zed Builds

Zed is my favorite code editor, but they aren't publishing binaries for Windows or Linux yet. So, I thought that I would. I'll try to keep this updated weekly until the official release. The Windows build requires Windows 11, and the Linux build should work with Ubuntu 22.04 LTS or later.


Troubleshooting:
If Zed won't launch on Windows, try updating your video drivers. You may need to install a few runtime dependencies on Linux.
